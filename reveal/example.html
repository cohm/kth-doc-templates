Heading hierarchy

Heading sizes are explicit at presentation scale, with strong contrast between levels:

H1 — slide title (92 px)

H2 — section heading (60 px)

H3 — subsection (44 px)

H4 — emphasis (34 px)

All sizes scale from --r-main-font-size: 42px. Headings use Figtree, body text uses Figtree as well.

Identity · Code

Syntax highlighting


import numpy as np
from kth_research import experiment, palette

KTH_BLUE = "#004791"

def measure(samples: int = 1024) -> np.ndarray:
    """Return a vector of measurements in KTH-friendly units."""
    rng = np.random.default_rng(seed=42)
    return rng.normal(loc=0.0, scale=1.0, size=samples)

if __name__ == "__main__":
    data = measure()
    fig  = experiment.plot(data, color=KTH_BLUE)
    fig.savefig("results.pdf")

Interactivity · Inline canvas

Animate anything

Drop a <canvas> into a slide and animate it with a script tag. The slide fires visibility events so you can start/pause animations as the user navigates.

  • fragmentshown · fragmenthidden
  • slidechanged · slidetransitionend
  • Reveal.getCurrentSlide()

Interactivity · Embedded widgets

Drop in any web app

Use data-src (not src) so reveal.js pauses off-screen widgets. Drop any Claude Design widget or web app into widgets/.
