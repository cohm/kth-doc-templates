Inline highlight

The .kthhl class renders a keyword in bold KTH blue — same visual as \kthhl{…} in LaTeX or <span class="kthhl"> in Markdown. Use it sparingly so it stays a signal, not noise.

Code with syntax highlighting


import numpy as np
from kth_research import experiment, palette

KTH_BLUE = "#004791"

def measure(samples: int = 1024) -> np.ndarray:
    """Return a vector of measurements in KTH-friendly units."""
    rng = np.random.default_rng(seed=42)
    return rng.normal(loc=0.0, scale=1.0, size=samples)

if __name__ == "__main__":
    data = measure()
    fig  = experiment.plot(data, color=KTH_BLUE)
    fig.savefig("results.pdf")

Interactivity

Inline canvas, embedded widgets, and arbitrary web apps — all live in your slides.

Inline canvas animation

Drop a <canvas> straight into a slide and animate it with a script tag. The slide fires visibility events so you can start/pause animations as the user navigates.

  • fragmentshown · fragmenthidden
  • slidechanged · slidetransitionend
  • Reveal exposes Reveal.getCurrentSlide()

Embedded widget · widgets/orbit.html

Use data-src (not src) so reveal.js can pause off-screen widgets to keep your CPU cool. Drop any Claude Design widget or web app into widgets/ and reference it the same way.
